# Bank App

This is a bank account app that allows you to desposit/withdraw money, see your balance, and see the the date and time of any tranactions.

# Setup

```
gem install bundler
bundle install
```

# To Use

```
irb
require './lib/account.rb'
account = Account.new
Call the methods on 'account'
```
